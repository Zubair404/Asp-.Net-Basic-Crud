<h2>ğŸš€ <b>Getting Started</b></h2>

<ol>
  <li>Clone this repository:
    <pre><code>git clone https://github.com/Zubair404/Asp-.Net-Basic-Crud.git</code></pre>
  </li>
  <li>Open the project in <b>Visual Studio</b> or <b>VS Code</b></li>
  <li>Update the connection string in <code>appsettings.json</code> with your SQL Server credentials</li>
  <li>Run the migrations (if using EF Core):
    <pre><code>dotnet ef database update</code></pre>
  </li>
  <li>Build and run the project:
    <pre><code>dotnet run</code></pre>
  </li>
  <li>Visit <code>https://localhost:5001</code> in your browser</li>
</ol>
